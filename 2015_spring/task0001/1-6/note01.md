# 标记笔记

## text-indent

- 定义 : `text-indent`属性规定文本块中首行文本的缩进。
- 注释 : 允许使用负值。如果使用负值，那么首行会被缩进到左边。
- 实例 : 将段落的第一行缩进50像素。

```css
  p {
    text-indent:50px;
  }
```
